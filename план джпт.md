Ты — senior frontend разработчик и UX/UI дизайнер уровня стартапов (Dribbble / Behance уровень). Твоя задача — создать полноценный современный туристический сайт.

# 🎯 ПРОЕКТ
Название: VIBE TRAVEL 03  
Регион: Республика Бурятия  
Цель: создать не просто сайт, а цифрового гида, который подбирает путешествия по "настроению" (вайбу)

Главная идея:
Пользователь не выбирает "экскурсии" → он выбирает ЭМОЦИЮ → получает маршрут

---

# 🔥 КОНЦЕПЦИЯ UX

Основной пользовательский сценарий:

1. Пользователь заходит на сайт  
2. Видит топ маршруты  
3. Далее выбирает вайб (настроение)  
4. Получает отфильтрованные маршруты  
5. Открывает маршрут  
6. Изучает и может сохранить в PDF  

---

# 🧠 ВАЙБ-СИСТЕМА (ОСНОВА)

Использовать как теги:

- ветер_в_лицо (адреналин)
- перезагрузка (релакс)
- кайфовый_ракурс (фото)
- место_силы (духовность)
- дешево_и_кайф (бюджет)
- в_дикую (природа)
- буузы_и_точка (гастро)
- приключение_началось (приключение)
- один_на_один (уединение)

---

# 🧩 СТРАНИЦЫ САЙТА

Обязательно реализовать:

1. Главная (index.html)
- hero секция (большой экран, градиент/фон)
- блок "топ маршруты"
- блок с выбором вайба (карточки)
- кнопка перехода к маршрутам

2. Страница маршрутов (routes.html)
- список карточек маршрутов
- фильтрация по вайбам
- фильтрация по длительности
- фильтрация по бюджету

3. Страница маршрута (route.html)
- название
- галерея (можно временно placeholder)
- описание
- список точек
- карта
- пасхалка
- кнопка "Сохранить в PDF"

4. Карта (map.html)
- точки маршрутов
- линии маршрутов
- при клике показывается название
