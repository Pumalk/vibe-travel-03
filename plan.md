## 📋 ИТОГОВЫЙ ПЛАН РЕАЛИЗАЦИИ «VIBE TRAVEL 03»

### Фаза 1: Каркас и данные (лёгкая)
- Инициализация структуры проекта (index.html, css/, js/, data/, assets/).
- Создание `routes.json` — 12 маршрутов с полями: id, название, теги (1-3 из 8), длительность, бюджет, сезон, координаты, массив точек с описанием и координатами, описание в молодёжном стиле, placeholder’ы для фото.
- Базовый HTML с контейнерами для SPA (без роутинга).
- CSS-переменные, сброс стилей, подключение шрифтов (локально).
- Favicon (юрта), текстовый логотип, прелоадер.
- **Результат:** пустая страница, данные загружены в консоль.

### Фаза 2: SPA-роутинг и основные страницы
- Простой роутер на `popstate` и `click`.
- Страницы: `home`, `routes`, `route-detail`, `events`, `map`.
- Рендеринг шапки (логотип, бургер-меню), подвала.
- Рендеринг Главной: Hero с параллаксом (статичное изображение + CSS-эффект), блок «Топ-подборки» (3 случайных маршрута из JSON).
- Страница Маршрутов: список карточек (без фильтров), каждая с динамическим градиентом по тегу.
- Страница Событий: карточки трёх событий с датами.
- Заглушка для Детальной страницы и Интерактивной карты.
- **Результат:** все страницы открываются, контент подгружается из JSON.

### Фаза 3: Вёрстка, адаптив и фильтры
- Mobile First: бургер-меню, сетки карточек (1/2/3 колонки).
- Фильтры на странице Маршрутов: чипсы для тегов + селекты (длительность, бюджет, сезон).
- Детальная страница: блоки подряд (описание, транспорт, советы), галерея (слайдер для моб., сетка + lightbox для десктопа), кнопки «Скачать PDF» и «Поделиться».
- Интерактивная карта (отдельная страница): Яндекс.Карты с кастомными маркерами, слоями (маршруты, достопримечательности, кафе, отели).
- На главной — превью-карта (интерактивная, с 3-5 точками).
- Кнопка «Построить маршрут» (2ГИС / Яндекс.Карты) на детальной.
- **Результат:** полностью работающий интерфейс, адаптивный, фильтры работают.

### Фаза 4: Базовый функционал и данные
- Избранное (localStorage), анимация сердечка.
- PDF-гид: стилизованный шаблон с логотипом, скриншотом карты (html2canvas), текстовыми координатами. Кнопка «Скачать».
- Кнопка «Поделиться» (Web Share API).
- Ленивая загрузка изображений (Intersection Observer).
- Валидация W3C, проверка читаемости кода.
- **Результат:** весь обязательный функционал работает.

### Фаза 5: Пасхалки и анимации
- **Пасхалка «Буузы»:** FAB-кнопка (долгое нажатие 1.5 сек / ховер), счётчик, случайное падение бууз (CSS-анимация), клик для удаления. Вибрация на мобильных.
- **Пасхалка «Дацан»:** на детальных страницах с дацаном — иконка с подсветкой, клик → вылет ленточек + надпись «Всё сбудется».
- Бесшовные переходы между страницами (CSS fade/slide).
- Фоновый орнамент (едва заметный паттерн).
- **Результат:** сайт с характером, приятные микроэмоции.

### Фаза 6: Документация и упаковка
- Пояснительная записка (PDF, 10-15 стр.) по шаблону конкурса.
- Файл `list.txt` с описанием контента (маршруты, фото, источники).
- Инструкция по запуску (`README.md`), включая настройку ключа Яндекс.Карт.
- Упаковка архива `Лебедев_Кирилл_Подборка_маршрутов_2.zip`.
- Финальное тестирование на реальных устройствах.
- **Результат:** готовый к отправке проект.
