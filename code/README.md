# VIBE TRAVEL 03
Проект для конкурса «Создай свой турсайт» (Бурятская ГСХА, 2026).

## 🚀 Запуск
1. Откройте `index.html` в браузере.
2. Для работы карт замените `YOUR_API_KEY` в файлах `index.html`, `route.html`, `map.html` на реальный ключ Яндекс.Карт (или используйте локальный сервер для тестов без ключа, если API позволяет).
3. Для корректной работы модулей лучше использовать локальный сервер (например, Live Server в VS Code).

## 📁 Структура
- `data/` - JSON с маршрутами и событиями.
- `src/css/` - Стили (BEM).
- `src/js/` - Логика (Vanilla JS).

## ✨ Фичи
- Эмоциональная навигация (Вайбы).
- Фильтрация по логике AND.
- Генерация PDF-гидов.
- Пасхалки и анимации.
