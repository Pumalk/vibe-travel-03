## 🔥 КОНЦЕПЦИЯ И ОСОБЕННОСТИ
Главная инновация — **эмоциональная навигация по «вайбам»**. Пользователь выбирает не тип маршрута, а желаемое настроение. Сайт становится цифровым гидом, подбирающим приключения по внутреннему состоянию.

**Обязательные вайб-теги (не менее 6):**
- `#кайфовый_ракурс` (фотолокации)
- `#цифровой_детокс` (релакс, природа)
- `#место_силы` (духовность, дацаны)
- `#ветер_в_лицо` (активный отдых, адреналин)
- `#буузы_и_точка` (гастрономия)
- `#в_дикую` (дикая природа, походы)

Каждый маршрут содержит в JSON явно заданное поле `gradient` (массив из двух цветов). Цвета варьируются на основе 5 базовых типов, вдохновлённых бурятской храмовой росписью, но не обязательно строго уникальны:
- Культура / духовность → красно-золотой
- Природа / релакс → зелёно-бирюзовый
- Мистика / энергия → фиолетово-розовый
- Гастрономия → оранжево-жёлтый
- Активный отдых → тёмно-синий

**Логика фильтрации:** все фильтры (вайбы, длительность, бюджет, сезон) применяются с логикой **AND**. Для вайбов это означает, что маршрут должен содержать **все выбранные теги**.

## 🧱 ОБЯЗАТЕЛЬНЫЕ СТРАНИЦЫ И ФУНКЦИОНАЛ

### 1. Главная (`index.html`)
- Hero-секция с лозунгом «Бурятия. Найди свой вайб» и фоновым градиентом/видео.
- Блок «Выбери настроение» — 6 интерактивных кнопок-вайбов, при клике сохраняют выбор в `localStorage` и ведут на `routes.html` с предустановленным фильтром.
- Топ-5 популярных маршрутов — карточки с градиентами из JSON.
- **Живая мини-карта** Яндекс с отключенными контролами, отображающая **все 12 маршрутов** (метки + полилинии). Инициализируется лениво при скролле.

### 2. Каталог маршрутов (`routes.html`)
- Сетка карточек 12 маршрутов: фото-плейсхолдер (цветная плашка с градиентом и текстом «Вставьте 1 фото»), название, вайб-теги (до 4-х), длительность, бюджет, кнопка «В избранное».
- Панель фильтрации: мультивыбор вайбов, длительности, бюджета, сезона. Логика AND.
- Сортировка по умолчанию, длительности, бюджету.
- Данные загружаются из `data/routes.json` через fetch.
- Избранное хранится в `localStorage` (массив ID).
- **Счётчик избранного в шапке сайта** (иконка сердечка с числом).
- На мобильных устройствах панель фильтров скрыта за кнопкой и выезжает шторкой (offcanvas).

### 3. Детальная страница маршрута (`route.html`)
- Работает через `?id=`. Если маршрут не найден — выводится сообщение «Маршрут не найден» и ссылка на каталог.
- Галерея: основное фото и миниатюры (плейсхолдеры).
- Название, живое описание, вайб-теги.
- Блок «Маршрут по дням» (расписание).
- Практическая информация: транспорт, бюджет, сезонность.
- **Яндекс.Карта** с точками маршрута (метки) и **полилинией, соединяющей точки в порядке их следования в массиве `points`**.
- Кнопка «Скачать PDF-гид». Документ должен содержать **обложку с названием и плейсхолдером для фото**, а также описание, программу и логистику (текстовый вариант, без скриншота карты).
- **Пасхалка №1:** в правом нижнем углу экрана — маленькая иконка кедра, при клике на которую появляется анимированный бурятский орнамент (например, SVG с анимацией).

### 4. Интерактивная карта (`map.html`)
- Полноэкранная Яндекс.Карта со всеми 12 маршрутами.
- Каждый маршрут отображается **уникальным цветом** полилинии и метками.
- При клике на метку — балун с названием и ссылкой на `route.html`.
- При загрузке карта автоматически масштабируется так, чтобы все маршруты были видны (`setBounds`).

### 5. Контент / Вайбы (`content.html`)
- Календарь событий из `data/events.json` (3 события: **Голос кочевников 2026**, **Сурхарбан 2026**, **Фестиваль Алтаргана** с реальными датами).
- Встроенные YouTube Shorts/видео о Бурятии (2-3).
- **Пасхалка №2:** при наведении на блок с видео появляется анимация волны (CSS-эффект).

### 6. Админ-панель (`admin.html`)
- **Страница не публикуется в меню.** Защищена паролем: при загрузке `prompt('Введите пароль')`. Пароль: `admin2026`. При неверном вводе — редирект на главную.
- HTML-форма с полями, соответствующими структуре маршрута.
- Поле для ввода путей к изображениям (текстовые пути).
- При сабмите генерируется JSON-объект маршрута и выводится в `<textarea>`.
- Кнопка «Копировать» копирует **весь массив маршрутов с добавленным/изменённым объектом**, чтобы можно было вставить в `routes.json`.

### 7. Избранное (`favorites.html`)
- Список избранных маршрутов (карточки), загружаемых по ID из `localStorage`.
- Возможность удаления.
- Если пусто — сообщение-заглушка.

## 🛠️ ТЕХНИЧЕСКИЙ СТЕК (СТРОГО)
- **HTML5** (семантические теги).
- **CSS3**: Grid, Flexbox, CSS-переменные, анимации. Методология **BEM**.
- **Vanilla JavaScript (ES6+)** — без фреймворков.
- **Библиотеки через CDN:**
  - Yandex Maps API (ключ в `data/config.js`, добавлен в `.gitignore`).
  - jsPDF (генерация PDF).
  - Feather Icons или Boxicons (иконки).
  - Шрифт **Inter** через Google Fonts (основной) + sans-serif fallback.
- **Данные:** `routes.json` (12 маршрутов) и `events.json` в папке `data/`.
- **Хранение:** `localStorage` для избранного и последнего выбранного вайба.
